# 🚀 Sentiment Analysis Dashboard (ML Powered)

A modern **Machine Learning powered Sentiment Analysis Web App** built using:

- 🧠 Logistic Regression (Scikit-learn)
- 📊 TF-IDF Vectorization
- 🌐 Flask Backend API
- 🎨 Interactive Dashboard UI

---

## ✨ Features

- Analyze text sentiment (Positive / Negative / Neutral)
- Real-time prediction using ML model
- Confidence score & sentiment score
- Upload CSV for batch analysis
- Interactive charts & dashboard
- Keyword extraction
- Clean modern UI

---

## 🧠 ML Model

- Algorithm: Logistic Regression
- Vectorizer: TF-IDF (Unigram + Bigram)
- Dataset: Sentiment140 (sampled)
- Accuracy: ~75–85% (depends on training size)
